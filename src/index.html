ngx-api-mimic (NgxApiMimic)

Angular API Mock library for demo apps. This library lets you mock data and create a fake API via an HTTP Interceptor inside Angular applications. It provides complete mock backend features: angular interceptor, router, controllers, decorators, and guards, with a NestJS-like architecture.

Key Features

  • Mock data and REST API simultaneously using @UsingSchema on Controllers
  • Intelligent schema-based mocked data generation via TypeScript interfaces
  • Automated mocked data storage in the browser's LocalStorage
  • Easily migrate mocked API to NestJS thanks to identical API decorators and architecture
