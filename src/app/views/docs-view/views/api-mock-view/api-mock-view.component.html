API Mock gives you all you need to mock API inside Angular app. From interceptor that redirects HTTP requests to functions inside the app, router that manages routing between controllers, controllers that work on specific path and contain methods to methods that work as specific endpoints serving specific HTTP method. API Mock Controllers are recreated to resemble parts of NestJS which makes it easier when you have Angular app with mock and you want to create API - you just copy-paste this code and write what's really inside.
