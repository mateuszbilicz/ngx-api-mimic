This basic example demonstrates how to create a simple controller and maintain a primitive state directly within the class properties.

Class code

At the core of our mock setup lies the controller class marked with the {{ '@Controller' }} decorator. Built on straightforward architectural patterns, it implements core endpoints via routing decorators such as {{ '@Get' }} and {{ '@Post' }}. State management is maintained natively via private class properties like the demonstrated counter.

Router and interceptor

To complete the lifecycle, the controller is wrapped within a mock router. Utilizing ngxApiMimicRouterFactory alongside a unique usePrefix isolates these API calls, effectively bridging Angular's backend proxy mapping.
